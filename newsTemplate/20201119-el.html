Γεια σου,
Για να γιορτάσουμε τα 20 χρόνια του GCompris, είμαστε στην ευχάριστη θέση να ανακοινώσουμε την έκδοση 1.0.

Αυτή η έκδοση περιλαμβάνει μια πολύ σημαντική λειτουργία: την προσθήκη, σε περισσότερες από 50 δραστηριότητες, της δυνατότητας 'Ρυθμίσεις δραστηριότητας' για επιλογή δεδομένων, που επιτρέπει να κάνουμε πολύ συγκεκριμένες επιλογές γι' αυτό που πρόκειται να μάθουμε σε κάθε δραστηριότητα.

Η νέα έκδοση περιλαμβάνει 4 νέες δραστηριότητες:

  • Την πολυαναμενόμενη δραστηριότητα 'Αναλογικός ηλεκτρισμός' που μας επιτρέπει να σχεδιάσουμε ηλεκτρικά κυκλώματα και να προσομοιώσουμε τη λειτουργία τους.
  • Τη δραστηριότητα 'Μαθαίνω αριθμούς' και δυο υποδραστηριότητες, 'Μαθαίνω πρόσθεση' και 'Μαθαίνω αφαίρεση'. Σκοπός είναι η διδασκαλία των αριθμών σε μικρά παιδιά, με εύκολες αριθμητικές πράξεις.
  • Στο 'Παιδικό πληκτρολόγιο', όταν πατάμε έναν χαρακτήρα αυτός εμφανίζεται στην οθόνη και ακούγεται ο αντίστοιχος ήχος, αν υπάρχει.
  • Η 'Βαρύτητα', που αντικαθιστά την παλιά 'Εισαγωγή στη βαρύτητα', εξηγεί καλύτερα την έννοια και είναι περισσότερο ρεαλιστική.

    • Θα βρείτε τα πακέτα αυτής της νέας έκδοσης για GNU/Linux, Windows και MacOS στη σελίδα λήψεων. Αυτή η ενημέρωση θα είναι σύντομα διαθέσιμη στο Android Play store, στο αποθετήριο F-Droid και στο Windows store. Λάβετε υπόψη, ότι το πακέτο για MacOS δεν έχει ακόμη επικυρωθεί.

    Μπορείτε να δείτε σε αυτή τη σελίδα την κατάσταση των αρχείων ήχου για τη γλώσσα σας: https://gcompris.net/voicestats/. Αν θέλετε, μπορείτε να βοηθήσετε προσφέροντας ηχογραφημένους τους ήχους της γλώσσα σας που διαπιστώνετε ότι λείπουν.

    Yπάρχουν 22 πλήρως υποστηριζόμενες γλώσσες: Βασκικά, Βρετονικά, Βρετανικά Αγγλικά, Καταλανικά, Καταλανικά Βαλένθιας, Παραδοσιακά Κινέζικα, Ολλανδικά, Γαλλικά, Ελληνικά, Εβραϊκά, Ουγγρικά, Ινδονησιακά, Ιταλικά, Μαλαγιάλαμ, Πολωνικά, Πορτογαλικά, Ρουμάνικα, Σλοβένικα, Ισπανικά, Σουηδικά, Τουρκικά, Ουκρανικά.

    Υπάρχουν επίσης 4 γλώσσες που δεν υποστηρίζονται πλήρως: Λευκορωσικά (85%), Πορτογαλικά Βραζιλίας (92%), Γερμανικά (89%) και Λιθουανικά (84%).

    Όπως είπαμε προηγουμένως, αποφασίσαμε να συμπεριλάβουμε τις γλώσσες που είναι τουλάχιστον κατά 80% μεταφρασμένες, έτσι ώστε τα παιδιά να έχουν καλύτερη εμπειρία και όχι ένα ημιτελές γλωσσικά λογισμικό. Δυστυχώς, είμαστε αναγκασμένοι να αφήσουμε εκτός τις εξής γλώσσες: Εσθονικά, Φινλανδικά, Ιρλανδικά Γαελικά, Σκωτικά Γαελικά, Γαλικικά, Χίντι, Μακεδονικά, Νέα Νορβηγικά, Ρωσικά, Σλοβακικά και Απλοποιημένα Κινεζικά.

    Γι' αυτό λοιπόν, αν η γλώσσα σας δεν είναι πλήρως μεταφρασμένη, ή δεν υποστηρίζεται καθόλου και θέλετε να βοηθήσετε, παρακαλώ επικοινωνήστε μαζί μας (support@gcompris.net) και εμείς θα σας κατευθύνουμε για να ξεκινήσετε τη μετάφρασή σας.

    Ένας ακόμη τρόπος για να βοηθήσετε, είναι να κάνετε μερικές αναρτήσεις στη δική σας κοινότητα σχετικά με το GCompris και μη διστάσετε να μας στείλετε τα σχόλιά σας.
