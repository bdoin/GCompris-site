As the title says, you can now buy the full version of GCompris for Windows 10 from the windows store.

This is a good way to make it more visible and easy to find for new users. Spread the word !

If you prefer to not use the store, or if you only want the free demo version, you can still download it from our Download page, and buy the activation code to unlock all the activities. The store is just one more way to distribute GCompris, and to provide some income to support the project.

As usual, the full version on Free-Software operating systems like GNU/Linux is free, but for proprietary operating systems like Windows, the full version has a cost. Of course, the source code of GCompris is and will always be under a Free-Software license.

Note: the package in the store contains the version using the software renderer instead of OpenGL, since it's they only way we have for now to make sure it will work on any computer. If you really want the version using OpenGL, get it from our Download page.
